# Harmonik Seri Ortalaması

Java diziler ile girilen sayının harmonik seri ortalamasını bulan program

Harmonik Ortalama formülü : n (eleman sayısı) / elemanların harmonik serisi

Harmonik Seri Formülü :

1 + (1/2) + (1/3) + (1/4) +.....+(1/n)
