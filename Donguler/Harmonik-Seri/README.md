# Harmonik Seri

Java ile girilen sayının harmonik serisini bulan program

Harmonik Seri Formülü :

`1 + (1/2) + (1/3) + (1/4) +.....+(1/n)`

**Senaryo**

```
Sayı giriniz: 2
1.5

```

```
Sayı giriniz: 49
4.4792053383294235
```
