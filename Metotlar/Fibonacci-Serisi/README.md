# Fibonacci Serisi

Recursive fonksiyon kullanarak kullanıcıdan alınan index değeriyle serinin o indexteki elemanını ekrana yazdıran program

**Senaryolar**

```
Fibonacci serisinin kaçıncı indisi:6
8

```

```
Fibonacci serisinin kaçıncı indexi:2
1
```
